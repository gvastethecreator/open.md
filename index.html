Ayuda

Atajos, flujo y trucos útiles

OpenMD está pensado para abrir, leer y cambiar de tema sin fricción. Esta pantalla reemplaza temporalmente la vista principal para tener una guía corta y sin ruido.

Atajos

F1
Abre o cierra esta pantalla de ayuda.
Esc
Vuelve a la vista anterior si la ayuda está abierta.
T
Cambia rápidamente al siguiente theme disponible.
Ctrl + + / -
Ajusta el zoom del contenido.
Ctrl + 0
Restablece el zoom al 100%.

Cómo usarlo

  1. Abre un archivo desde el botón Abrir o arrástralo a la ventana.
  2. Lee el documento con render Markdown, bloques de código y Mermaid.
  3. Usa enlaces relativos entre archivos para navegar documentación local.
  4. Si sueltas varios archivos, el primero se abre aquí y los demás van a nuevas ventanas.

Qué admite

  • Archivos .md, .markdown y .txt.
  • Imágenes locales relativas dentro del mismo árbol de carpetas.
  • Temas integrados para lectura clara en fondos claros u oscuros.
  • Diagramas Mermaid embebidos dentro del Markdown.

Tips rápidos

  • Si vas a revisar documentación local, deja esta ayuda a mano con F1.
  • El estado superior te muestra el archivo activo o si estás en modo ayuda.
  • Los links externos se abren fuera de la app para no romper el contexto de lectura.
